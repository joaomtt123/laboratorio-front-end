O elemento HTML < sub > define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O

A tag < sub > existe desde as primeiras versões do HTML, sendo parte do HTML 2.0, lançado em 1995. Desde então, ela vem sendo mantida como uma forma semântica de representar texto subscritos, com foco principalmente em publicações científicas e acadêmicas.
